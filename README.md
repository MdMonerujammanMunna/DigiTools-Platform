## Overview
DigiToolsHub is a modern, responsive single-page app built with **React.js**. The platform delivers a fast, smooth, and engaging user experience with dynamic product browsing and seamless interactions.

---

## Smart Cart System
- Real-time updates with instant UI refresh  
- Live cart count visible in the navbar  
- Auto-calculates total pricing  
- Smooth checkout process with confirmation  
- Cart clears after successful purchase  
- Simple and clean empty cart experience  

---

## Dynamic Product Cards
- Responsive grid layout for all screen sizes  
- Includes detailed info like name, description, and pricing  
- Highlights key features with clean bullet points  
- Shows attractive labels like **Popular**, **Best Seller**, **New**  

---

## Smart Notifications
- Stylish toast messages using **React Toastify** for add/remove actions and order confirmation
